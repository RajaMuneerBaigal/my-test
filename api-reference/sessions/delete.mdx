---
title: "Delete Session"
api: "POST https://app.chatsistant.com/api/v1/session/{uuid}/delete"
description: "Delete a session by its uuid"
---

### Path

<ParamField path="uuid" type="string" required>
  Session uuid
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if API call was successful
</ResponseField>

<RequestExample>

```bash Curl
curl --location --request POST 'https://app.chatsistant.com/api/v1/session/{uuid}/delete' \
--header 'Authorization: Bearer <token>'
```

```py Python
import requests

uuid = '<message-uuid>'
url = f'https://app.chatsistant.com/api/v1/session/{uuid}/delete'
headers = {
    'Authorization': 'Bearer <token>'
}

response = requests.POST(url, headers=headers)

if response.status_code == 200:
    print("Request successful! Session deleted.")
else:
    print("Request failed with status code:", response.status_code)
    print(response.text)
```

```ts JavaScript
const axios = require("axios");

const uuid = "<message-uuid>";
const url = `https://app.chatsistant.com/api/v1/session/${uuid}/delete`;
const headers = {
  Authorization: "Bearer <token>",
};

axios
  .post(url, { headers })
  .then((response) => {
    console.log("Request successful! Session deleted.");
  })
  .catch((error) => {
    console.error("Request failed:", error);
  });
```
